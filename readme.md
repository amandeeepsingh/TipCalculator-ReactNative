# Steps to create and run React Native Apps

* ` brew install node `
* ` brew install watchman `
* ` npm install -g react-native-cli `
> Install Xcode (for iOS only)
* ` react-native init TipCalculator `
* ` cd TipCalculator/ `
* ` react-native run-ios `
